## ğŸš€ **Getting Started**

### **Prerequisites**
- Node.js 18+ 
- npm or yarn
- Modern web browser

### **Installation**
```bash
# Clone the repository
git clone https://github.com/punam06/PTE-mobileApp.git

# Navigate to the app directory
cd PTE-mobileApp/pte-mobile-app

# Install dependencies
npm install

# Start development server

```
